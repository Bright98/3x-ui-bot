# 3x-ui bot

This bot tells clients traffic usage by config url.

It works for `vless` and `vmess` configs now.

## how to run:
- `git clone https://github.com/Bright98/3x-ui-bot.git`
- `cd 3x-ui-bot`
- change `.env` file:
  - bot token
  - server ip
  - panel port
  - panel username
  - panel password

- `sh install.sh`
